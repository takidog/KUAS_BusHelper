# 架設

Python版本3.5 ~3.6

1. 設定Config.cfg
2. `python main.py`



**Config.cfg**

需要申請一個Telegram Bot API 這裡不再細說註冊流程

```
[KUAS_Default]
account = ''
password = ''

[bus_day]
hahaha = 0

[Debug]
log = 0
debug_mode = 0

[Telegram]
API = XXXXX:XXXXXXXXXX
```



**Setup**



```bash
cd KUAS_BusHelper\src

python main.py
```
